## Daily Statistics

At a glance, see:

- **Total Active Time** - Hours spent at your computer
- **Sessions** - Number of work sessions
- **Screenshots** - Total captures for the day

## Application Usage

### Pie Chart

Visual breakdown of time spent by application. Hover over segments to see exact durations.

### Usage Table

Sortable list showing:

- Application name
- Time spent
- Percentage of total time

## Activity Patterns

### Hourly Distribution

Chart showing when you're most active throughout the day. Useful for identifying your peak productivity hours.

### Activity Heatmap

A 7-day by 24-hour grid showing your typical activity patterns. Darker cells indicate more activity at that time slot.

## Time Periods

Switch between views:

- **Today** - Current day's activity
- **Week** - Last 7 days aggregated
- **Month** - Last 30 days aggregated

## Top Windows

See which specific windows (not just applications) you spent the most time in. Useful for understanding if you're focused on specific projects or documents.
