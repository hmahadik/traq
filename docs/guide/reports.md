# Reports

Generate activity reports for timesheets, standups, or personal records.

## Creating a Report

1. Go to **Reports** in the navigation
2. Select a time range using natural language or the date picker
3. Choose a report type
4. Click **Generate**

## Natural Language Time Ranges

Traq understands phrases like:

- `today`, `yesterday`
- `this week`, `last week`
- `past 3 days`, `last 7 days`
- `this month`, `last month`
- `January 2024`, `Q1 2024`

## Report Types

### Summary Report

High-level overview including:

- Total active time
- Session count
- Top applications
- Key activities

### Detailed Report

Comprehensive report with:

- All sessions with summaries
- Application usage breakdown
- Hour-by-hour activity
- Screenshots (optional)

### Standup Report

Formatted for team standups:

- What you worked on
- Key accomplishments
- Time breakdown by project/app

## Export Formats

- **Markdown** - For docs or notes apps
- **HTML** - Styled, shareable web page
- **PDF** - Printable document
- **JSON** - Structured data for processing

## Report History

Previously generated reports are saved and accessible from the Reports page. You can:

- View past reports
- Re-export in different formats
- Delete old reports
