# Timeline View

The Timeline is your daily activity log, showing sessions and screenshots in chronological order.

## Calendar Heatmap

The calendar view shows activity intensity across days:

- **Darker cells** = more activity
- Click any day to view that day's sessions
- Navigate between months using the arrows

## Session Cards

Each session displays:

- **Time range** - When the session started and ended
- **Duration** - How long the session lasted
- **Screenshot count** - Number of screenshots captured
- **Top applications** - Most-used apps during the session
- **AI Summary** - Natural language description (when AI is configured)

## Browsing Screenshots

Click on a session to expand it and see:

- Screenshot thumbnails in chronological order
- Click any thumbnail for a full-size view
- Use the lightbox to navigate between screenshots

## Session Details

Click "View Details" on a session card to see:

- All screenshots in a gallery view
- Activity log with window focus events
- Time spent per application within the session

## Navigation Tips

- Use the date picker to jump to a specific date
- "Today" button returns to current day
- Previous/Next arrows move between days
